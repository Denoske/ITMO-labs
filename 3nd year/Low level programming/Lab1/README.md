## О проекте
Лабораторная работа посвящена написанию программе, которая способна работать с одним файлом весом 10 гб. 

По варианту реализован файл содержащий данные в виде документного дерева.

## Сборка

  bash$ gcc main.c -o main

## Запуск

  bash$ ./main
